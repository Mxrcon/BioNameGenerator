TOPIC
    BioNameGenerator

SHORT DESCRIPTION
    This module provides functions for Random Generation of Words or Phrases 
    using Dictionaries inspired by Biology and Science.

LONG DESCRIPTION
    The function `Get-RandomKeyword` provides a method of generating random
    Keywords from one of the dictionaries packed with the Module in a SQLite
    database.

    `Get-RandomPhrase` generates random phrases with 1 or more words acessing the
    dicionaries multiple times, and a delimiter can be specifed for the concatenation
    of the output.
